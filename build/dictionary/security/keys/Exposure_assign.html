Exposure assign (object-based)
Description
Permission to assign an object of type Exposure. Does nothing if the object is new, checks expraown if reassigning an object the user owns, and checks expraunown if reassigning an object that the user doesn't own.
Related Permissions

expraown
Permission to reassign your own exposures

expraunown
Permission to reassign exposures owned by other users
