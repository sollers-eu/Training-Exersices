claimview (object-based) - View claims
Description
General permission to view a claim
Roles

Adjuster
Base permissions for an adjuster

Claims Supervisor
Base permissions for a claims supervisor

Clerical
Base permissions for a clerical user

Customer Service Representative
Limited view for a customer service representative

Integration Admin
Permissions for integration admin

Manager
Base permissions for a manager

New Loss Processing Supervisor
Base permissions for a claims supervisor

Reinsurance Manager
Base permissions for a reinsurance manager

Superuser
Superuser with full permissions

Underwriter
Base permissions for an underwriter

Viewer
User with view-only permissions

ClaimInfo view

Permission to view an incident

Note view
Permission to view a note. A note can only be viewd by a user who has the noteview permission.
