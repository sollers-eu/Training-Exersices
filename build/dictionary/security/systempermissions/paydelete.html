paydelete (object-based) - Delete payments
Description
Permission to delete payment transactions
Roles

Adjuster
Base permissions for an adjuster

Claims Supervisor
Base permissions for a claims supervisor

Clerical
Base permissions for a clerical user

Manager
Base permissions for a manager

New Loss Processing Supervisor
Base permissions for a claims supervisor

Superuser
Superuser with full permissions

Related Application Permission Keys

Check delete
Permission to delete a check on a claim. Checks delete payment or delete manual payment depending on the type of check

Payment delete
Permission to delete a payment on a claim. Checks delete payment or delete manual payment depending on the type of check the payment is on

Payment deletenormal
Permission to delete a normal payment on a claim
