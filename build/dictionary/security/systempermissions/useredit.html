useredit (object-based) - Edit users
Description
Permission to edit an existing user, except for roles, authority limits, or attributes
Roles

Superuser
Superuser with full permissions

User Admin
Permissions for user admin

Related Application Permission Keys

Contact editlocal
Permission to edit a Contact entry. A user can edit an contact entry if they have edit contact permission and it's a normal contact. If it's a user contact, they must also have edit user permission.

Credential edit
Permission to edit an existing credential. Note that a user can always edit their own password

User edit
Permission to edit an existing user
