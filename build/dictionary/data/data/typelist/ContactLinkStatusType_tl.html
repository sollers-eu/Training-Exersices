ContactLinkStatusType     (cctl_contactlinkstatustype)
Represents the link status of a Contact with its associated Address Book Contact.
Attributes: Final

Typecodes
Code Name Description Priority Internal Retired
IN_SYNC In sync The Contact is in sync with the associated Address Book Contact.   true  
NOT_FOUND Not found No associated Address Book Contact found.   true  
OUT_OF_SYNC Out of sync The Contact is out of sync with the associated Address Book Contact.   true
