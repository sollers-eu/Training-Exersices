RecoveryTAccountLineItem (cc_recoverytaccountlineitem) (delegates to Editable, Extractable, TAccountLineItemDelegate)
Description
The entry of a specific amount of money, either crediting or debiting one T-account. A TaccountTransaction will contain a pair of balancing TAccountLineItems - a debit to one T-account and an equal credit to another.
This entity is overwritten by the application during staging.

DenormTransaction  foreign key to RecTAccountTransaction     (database column: DenormTransactionID)    (loadable)?    (overwritten on load)?
Denormalized FK to RecTAccountTransaction table in order to speed up certain queries. If both CreditingTransactionID and DebitingTransactionID are not null, then this field is NULL, otherwise this field will have same value as the non-null FK. This allows us to query against this field only when looking for RecTAccountTransactions that are currently contributing to a RecoveryTAccount.
