Evaluation (cc_evaluation) (delegates to ClaimContactRoleOwner, Extractable, Retireable)
Description
Represents the liabilities and damages in a Claim's plan of action. A plan of action is an agreement between parties that concludes a discussion of outstanding liabilities. The Evaluation object brings together key contacts and financial data related to a claim.
