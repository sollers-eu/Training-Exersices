TAccountLineItem (cc_taccountlineitem) (delegates to Extractable, Retireable, TAccountLineItemDelegate)
Description
The entry of a specific amount of money, either crediting or debiting one T-account. A TaccountTransaction will contain a pair of balancing TAccountLineItems - a debit to one T-account and an equal credit to another.
This entity is overwritten by the application during staging.

DenormTransaction  foreign key to TAccountTransaction     (database column: DenormTransactionID)    (loadable)?
Denormalized FK to TAccountTransaction table in order to speed up certain queries. If both CreditingTransactionID and DebitingTransactionID are not null, then this field is NULL, otherwise this field will have same value as the non-null FK. This allows us to query against this field only when looking for TAccountTransactions that are currently contributing to a TAccount.
