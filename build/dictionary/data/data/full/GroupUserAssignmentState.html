GroupUserAssignmentState (cc_groupuser_assign) (delegates to Versionable)
Description
Joins a GroupAssignmentState entity to one or more Users. Also stores per user load information for round-robin assignment. This is an internal class which should only be used by the internals of the assignment engine.
