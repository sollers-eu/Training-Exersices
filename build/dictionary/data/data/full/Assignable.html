Assignable (Virtual)
Description
  • The assignment status ("Assigned", "Unassigned" and so on)
  • The current assignee - a queue or user and their corresponding group
  • The previous assignee - a queue or user and their corresponding group
  • The user who last assigned the entity
  • The time when the entity was last assigned
If an entity is assigned to a queue then the assigned group will be the queue's group. If an entity is assigned to a user then the assigned group will be one of the user's groups. Do not assign an entity to both a user and a queue. If an entity is assigned to a group it will usually also be assigned to a user or queue, though while assignment rules are running it may temporarily be assigned to just a group.
