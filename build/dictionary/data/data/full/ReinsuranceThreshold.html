ReinsuranceThreshold (cc_reinsurancethreshold) (delegates to Retireable)
Description
This reference table contains the information necessary to determine whether a claim has exceeded the reinsurance reportable threshold notification limit based upon the claim's policy type, coverage types and loss causes.
