ReserveLineWrapper (cc_reservelinewrapper) (delegates to Extractable, Versionable)
Description
Provides a level of indirection between a BulkInvoiceItem and its ReserveLine. This is necessary when a BulkInvoiceItem is created for a non-existent reserve line, and so cannot be committed to the database. The BulkInvoiceItem has a non-null foreign key (FK) to ReserveLineWrapper, with a second FK from the wrapper to the actual reserve line that is null if the reserve line doesn't yet exist. The Bulk Invoice screen displays the wrapper's correct but uncreated reserve line so the user never knows if the displayed reserve line exists yet.
