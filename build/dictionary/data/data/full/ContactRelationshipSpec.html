ContactRelationshipSpec (Virtual) (delegates to Versionable)
Description
<IncludeRelationship> in the contact-sync-config.xml is called "Syncable Relationship". A ContactRelationshipSpec contains two sets of syncable relationships, "SourceRelationships" and "TargetRelationships" which is grouped according to the contactBidiRel. This entity has been deprecated as of ClaimCenter 8.0 along with the IAddressBookAdapter plugin.
