SecurityZone (cc_securityzone) (delegates to Retireable)
Description
Used to partition groups, users and entities assigned to them. By default, a user in one zone cannot view an entity assigned to someone in a different zone. Each group belongs to exactly one security zone, and a customer can define an arbitrary number of security zones. A user is a member of a zone if they are a member of a group in the zone, and an assignable entity belongs to a zone if its assigned group belongs to the zone. For full details see the Administration Guide.
