ContactAutoSyncWorkItem (cc_autosyncworkitem) (delegates to EventAware, KeyableBean, WorkItem)
Description

Entity: ContactAutoSyncWorkItem
Attributes: Extendable?, Final?, Keyed?, Sourceable?
