OfficialID (cc_officialid) (delegates to Extractable, OverlapTable, Retireable)
Description
Represents the pairing of a given official ID number and a jurisdiction. Examples of these would include the pairing of a state and a drivers license number or state tax ID number. Multiple OfficialIDs can be associated with a given Contact.
