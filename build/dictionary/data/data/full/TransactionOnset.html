TransactionOnset (cc_transactiononset) (delegates to Extractable, Versionable)
Description
A TransactionOnset relates a transaction to its onsetting transactions (if any). It contains two links, one to the original transaction and one to the new onset transaction. A TransactionOnset is created whenever a user recodes or transfers a payment in ClaimCenter. In the case of a recode or transfer the onset payment has the same line items as the original payment but a different ReserveLine coding.
