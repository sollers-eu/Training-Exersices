ClaimSnapshot (cc_claimsnapshot) (delegates to Extractable, Retireable)
Description
Captures the first notice of loss (FNOL) data for a Claim. A ClaimSnapshot stores all the data for an FNOL as text string. This text contains an XML description. The format of the XML is version-dependent -- meaning there may be XML structure differences between ClaimCenter versions.
Entity: ClaimSnapshot
Attributes: Editable?, Extendable?, Final?, Keyed?, Sourceable?, Versionable?
