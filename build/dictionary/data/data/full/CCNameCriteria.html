CCNameCriteria (Virtual) (delegates to Versionable)
Description
A component used as part of a larger search criteria entity. Encapsulates the information entered by the user to restrict the search to particular contact, specified by some combination of name, tax ID and official ID. One CCNameCriteria entity will be used for each field that needs to be restricted. For example if a search criteria was being constructed to specify searches for an entity which had two different contact fields, then the search criteria would likely contain two distinct CCNameCriteria components. This ClaimCenter specific CCNameCriteria entity differs from the platform NameCriteria entity in that it is extendable and can be used with configurable search
