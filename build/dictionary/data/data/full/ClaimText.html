ClaimText (cc_claimtext) (delegates to Extractable, Versionable)
Description
Stores long text descriptions associated with a Claim. For example, a medical diagnosis for a Workmen's Compensation claim is type of ClaimText. Changes to these text descriptions triggers validation.
