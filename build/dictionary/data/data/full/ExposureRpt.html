ExposureRpt (cc_exposurerpt) (delegates to Extractable, Retireable)
Description
Denormalized financial calculations for exposures. These values are kept up-to-date as financials are recorded in the system. They can be recalculated if necessary through a batch process when the server is in maintenance mode.
Entity: ExposureRpt
