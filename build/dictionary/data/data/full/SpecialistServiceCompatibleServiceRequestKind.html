SpecialistServiceCompatibleServiceRequestKind (cc_specsvccompsvcreqkind) (delegates to Versionable)
Description
The presence of an instance of this entity indicates that the related SpecialistService can be used on Service Requests with the specified Kind. In the base configuration, instances of this entity must only be linked to leaf SpecialistServices.
