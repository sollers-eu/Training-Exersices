ContactUpdateResult (Virtual) (delegates to Versionable)
Description
ContactUpdateResult has been deprecated as of ClaimCenter 8.0 along with IAddressBookAdapter.. ContactUpdateResult represents the result of creating or updating a Contact. This operation can either succeed or fail. If it failed, then errors will be accessible via getValidationResult. If it succeeded, then there should be no errors in the ValidationResult, and the created or updated Contact will be accessible via getContact.
