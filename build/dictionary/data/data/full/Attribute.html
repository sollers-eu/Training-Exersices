Attribute (cc_attribute) (delegates to Retireable)
Description
Describes a name-value pair. An administrator can create an Attribute entity to represent custom user characteristics, for example, Spanish. Then, the administrator can associate the Attribute with one or more system users. Studio users can reference an Attribute from Studio when writing rules.
