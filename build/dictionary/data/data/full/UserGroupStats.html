UserGroupStats (cc_usergroupstats) (delegates to KeyableBean)
Description
Represents the combined statistics for a given user in a group. This entity aggregates claim, exposure, and activity statistics. Refer to the DashboardStats entity for statistics displayed for a user or in the dashboard.
Entity: UserGroupStats
Attributes: Final?, Keyed?, Sourceable?
