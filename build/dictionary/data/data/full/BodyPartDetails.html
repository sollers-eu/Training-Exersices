BodyPartDetails (cc_bodypart) (delegates to Extractable, Retireable)
Description
Collects details of body injury for an InjuryIncident. Each instance of this entity represents a body part injured in the incident. An individual InjuryIncident can have multiple BodyPartDetails, representing injured body parts of the Person (ClaimContact) with role of "injured" for the InjuryIncident.
