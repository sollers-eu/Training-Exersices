BIValidationAlert (cc_bivalidationalert) (delegates to Retireable)
Description
Represents one bulk invoice validation alert message. A bulk invoice validation plugin constructs one or more instances of this entity when it encounters validation errors. Each instance is related to a single BulkInvoice and has both an alert type and a message indicating the reason for the validation failure.
