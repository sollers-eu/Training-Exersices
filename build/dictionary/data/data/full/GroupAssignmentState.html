GroupAssignmentState (cc_group_assign) (delegates to LastAssignedUserContainer, LastAssignedUserContainerBase, Retireable)
Description
Maintains round-robin assignment state for a particular group or for a particular group and all its subgroups. This is an internal class which should only be used by the internals of the assignment engine.
Entity: GroupAssignmentState
Attributes: Editable?, Extendable?, Final?, Keyed?, Sourceable?, Versionable?
Concrete FK references:
