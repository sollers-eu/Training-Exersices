Credential (cc_credential) (delegates to Retireable)
Description
Maintains a user's system level-security identification. Each user has a corresponding Credential object. This entity maintains information related to authorization and authentication on the system. A Credential holds both the user's password and controls whether or not the user is active or not. This entity also tracks login attempts.
