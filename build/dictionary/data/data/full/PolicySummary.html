PolicySummary (Virtual) (delegates to Versionable)
Description
Non-persistent entity containing simplified information about a Policy. This object is returned in the result set during a Policy search (IPolicySearchAdapter.searchPolicies()), giving the user some context with which to select the desired policy for full retrieval.
Entity: PolicySummary
Attributes: Exportable?, Extendable?, Final?, Keyed?, Sourceable?
